NexusHub API (0.1.0)

Download OpenAPI specification:

URL: https://github.com/tomeksdev/NexusHub License: MIT

REST API for NexusHub — a WireGuard VPN management dashboard with PostgreSQL, JWT auth, and eBPF-backed security rules.

Conventions

  • All endpoints live under /api/v1.
  • Authenticated routes require Authorization: Bearer <access_token>.
  • List endpoints accept ?limit=, ?offset=, and ?sort= (prefix with - for descending) and return a PageEnvelope.
  • Errors use the Error schema with a stable code field — rely on that rather than on error copy.
  • Every response echoes X-Request-ID. Clients may send their own (must be a UUID) or let the server generate one. X-Correlation-ID is only echoed when the caller supplied it.

Server-Sent Events stream of live peer state

Emits one snapshot event on connect with the full peer list, then one peer event per poll tick per peer whose handshake time or byte counters changed, plus a ping event every ~15s as a keepalive. The server polls the kernel every 5s by default; each SSE event's data: line is a JSON object matching PeerEvent.
